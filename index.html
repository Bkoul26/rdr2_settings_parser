RDR2 settings.xml Parser (a work in progress!)

RDR2 for PC has a lot of settings. Taking screenshots is clunky, so why not parse that settings file to a nice text block?

Paste the contents of your RDR2 settings file (found at /Users/Username/Documents/Rockstar Games/Red Dead Redemption 2/Settings/system.xml) in the left area,
hit the button, and a nicely formatted settings block will appear on the right, ready to copy/paste into your favorite online discussion.

  • For feedback, see this reddit thread or Github .
  • The game only saves your settings to system.xml on succesful exit.
  • I'm following the in-game order of settings as closely as possible.
  • If you like what I do, buy me a coffee more games.

Update: 07/11 4:47 GMT: Everything currently parsed correctly for the basic (non-locked) settings. Working on advanced settings, but there are some of them that don't map to an in-game slider.
